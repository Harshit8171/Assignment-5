**Olympics SQL Analysis**

Project Overview

This project contains SQL queries performed on the Olympic Athlete Dataset to analyze athlete participation, medal distribution, and country performance.

The project focuses on advanced SQL concepts such as CASE statements, GROUP BY, HAVING, and conditional aggregation.

**Dataset**

**Table Used:** athlete_olympics

Main Columns:

Name

Sex

Age

Height

Weight

Team

NOC (Country Code)

Year

Season

Sport

Event

Medal

**Key Analysis**

Medal classification of countries using CASE

Athlete career longevity analysis

Medal-per-athlete efficiency by country

Dominant sports by country

Female participation percentage by year

Youngest gold medalists by sport

Multi-sport athlete identification

Countries winning medals but no gold

SQL Concepts Used

CASE Statements

GROUP BY

HAVING

Aggregate Functions

Conditional Aggregation
